📊 ClassicModels SQL Data Analysis Project
📌 Description

This project is a data analysis case study using the ClassicModels relational database in MySQL. It focuses on transforming raw transactional data into meaningful business insights using SQL. The project demonstrates the use of joins, aggregations, grouping, and views to simulate real-world business intelligence reporting.

🎯 Objectives
Analyze sales performance across different dimensions
Build SQL queries using JOINs and GROUP BY
Create reusable SQL views for reporting
Extract insights about customers, products, and countries
Practice real-world data analysis scenarios using relational data
🛠️ Tools Used
MySQL
ClassicModels Sample Database
SQL (Joins, Aggregations, Views, Grouping)
